# 字幕翻译技能

## 触发条件
当用户要求翻译 SRT 字幕、生成中英双语字幕、字幕翻译时使用此技能。

## 输入格式

用户输入为标准 SRT 字幕文件，内容可能是任意语言（中文、英文、日文、韩文等）

每条字幕包含：
- 序号
- 时间轴
- 原始文本

## 输出格式（必须严格遵守）

保持原有 SRT 结构完全不变：
- 序号不变
- 时间轴不变

每条字幕输出为**两行文本**：

```
中文翻译（如原文不是中文）/ 或中文原文（如原文是中文）
英文翻译（如原文不是英文）/ 或英文原文（如原文是英文）
```

## 双语规则（核心逻辑）

根据原文语言自动判断：

### 情况 A：原文是中文

```
中文（原文）
英文翻译
```

### 情况 B：原文是英文

```
中文翻译
英文（原文）
```

### 情况 C：原文是其他语言（如日文、韩文等）

```
原文
中文翻译（合并为一行，或原文 + 中文翻译）
```

注：情况 C 中，如果内容允许，优先输出原文 + 中文翻译两行；如超出字符限制可只保留中文翻译。

## 翻译要求

### 英文翻译：
- 自然、口语化、符合语境
- 避免中式英语，优先地道表达

### 中文翻译：
- 通顺、易懂、符合中文表达习惯
- 根据上下文补全省略信息（如主语、语气）
- 保持语气一致（演讲 / 对话 / 解说等）

## 字幕规范

- 单行尽量简洁，避免过长
- 英文建议控制在 **42 字符以内**
- 若过长可合理断行（但保持在该字幕块内）
- 避免无意义断句

## 一致性

- 专有名词、术语前后一致
- 技术内容优先使用行业通用表达
- 人名、品牌名保持统一翻译

## 禁止行为

- 不要修改时间轴
- 不要删除或合并字幕编号
- 不要添加解释、注释或额外说明
- 不要输出除 SRT 以外的内容

## 输出要求

- 仅输出最终 SRT 内容
- 不要添加任何说明文字
- 直接输出可使用的双语字幕文件
