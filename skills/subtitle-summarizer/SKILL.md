# 视频内容策划与摘要技能

## 触发条件
当用户要求从字幕生成视频摘要、标题、简介、关键词时使用此技能。

## 输入格式

用户输入为 SRT 字幕文件（包含时间轴与文本）或者纯文本
字幕、文本可能为任意语言

## 处理要求

- 自动理解字幕/文本内容（无需逐句翻译）
- 提取核心主题、关键信息和内容重点
- 判断视频类型（如：技术讲解 / 演讲 / 教程 / 访谈 / 新闻等）
- 如为技术类内容，优先保留专业术语

## 输出内容（共三部分）

### （1）标题（Title）

- 一句话总结核心内容
- 简洁、有吸引力
- 可适当优化表达（偏"内容平台风格"，但不要夸张或标题党）

### （2）简介（Description）

- 2～4 句话
- 概括视频讲了什么 + 有什么价值
- 语言自然流畅，信息密度高

### （3）关键词（Keywords）

- 提取 5～10 个关键词
- 使用「#」连接
- **仅一行输出**，例如：

```
#AI #机器学习 #大模型 #创业 #OpenAI
```

## 语言要求

- 默认输出为**中文**
- 专有名词（如技术术语、人名、产品名）可保留原文

## 质量要求

- 不要逐字复述字幕
- 要"提炼"，不是"拼接"
- 信息准确，不杜撰
- 技术内容要严谨，避免误解

## 禁止行为

- 不要输出与内容无关的信息
- 不要添加解释性说明（如"以下是总结"）
- 不要输出多余格式或标记

## 输出格式（严格遵守）

```
标题：xxx
简介：xxx
关键词：#xxx #xxx #xxx
```
