# 图像水印技能

## 触发条件
当用户要求为图片添加水印、Logo 或进行图片标注时使用此技能。

## 前置要求
- Python 3.x 已安装
- Pillow 库已安装（`pip install Pillow`）

## 工作流程

### 1. 收集必要信息
需要用户提供以下信息：
- **原图路径**：需要添加水印的图片文件
- **水印图路径**：PNG 格式水印图片（建议透明背景）
- **位置**：5 个固定位置可选
  - `top-left`：左上角
  - `top-right`：右上角
  - `center`：居中
  - `bottom-left`：左下角
  - `bottom-right`：右下角
- **可选参数**：
  - 透明度（默认 0.5）
  - 边距像素（默认 20）
  - 缩放比例（默认 1.0）
  - 输出路径（默认原图路径后加 `_watermark`）

### 2. 确认参数
向用户确认所有参数是否正确，特别是位置选择。

### 3. 执行水印处理
运行水印脚本处理图像：
```bash
python watermark.py -i <原图路径> -w <水印图路径> -p <位置> -o <输出路径> [--opacity 0.5] [--margin 20] [--scale 1.0]
```

### 4. 输出结果
- 成功：告知输出文件路径，可显示处理前后对比
- 失败：显示错误信息，引导用户检查文件或路径

## 注意事项
- 水印图片建议使用 PNG 格式，支持透明通道
- 建议透明度设置在 0.3-0.7 之间，避免遮挡原图内容
- 边距根据水印大小调整，避免过于贴近边缘
- 处理前备份原图（脚本会自动保留原文件）
- 如果水印太大，可建议用户缩小水印图或调整缩放比例
