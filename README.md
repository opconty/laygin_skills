# laygin skills introduction
> 用于存放 公众号（laygin）文章 提及的agent skills

## 当前技能一览

| 技能名称 | 功能 | 触发词 | 分类 |
|---------|------|--------|------|
| content-extractor | 从长字幕/文本提取高价值片段和核心观点 | "提取精华"、"高光片段"、"内容提炼" | 内容处理 |
| cover-prompt-generator | 自动生成封面图提示词（自动匹配 6 种风格） | "生成封面提示词"、"封面图建议" | 图像生成 |
| image-watermark | 为图像添加图片水印/Logo | "加水印"、"给图片加水印" | 图片处理 |
| illustration-prompt-generator | 生成 5 条以上高质量文生图提示词 | "生成配图提示词"、"配图建议" | 图像生成 |
| ltx-prompt-generator | 生成 LTX-2.3 视频提示词（文本→视频） | "生成视频提示词"、"LTX prompt" | 视频生成 |
| subtitle-summarizer | 从字幕提取核心信息，生成标题、简介、关键词 | "视频摘要"、"生成标题"、"内容摘要" | 内容处理 |
| subtitle-translate | 将任意语言 SRT 转换为中英双语字幕 | "翻译字幕"、"SRT 翻译" | 字幕处理 |
| weekly-report | 自动生成周报（数字编号格式） | "写周报"、"生成周报" | 文档生成 |
